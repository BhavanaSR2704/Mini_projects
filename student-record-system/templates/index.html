🔍 Linear Search

Scans every record one by one.

O(n)

⚡ Binary Search

Sorts via Quick Sort, then searches.

O(n log n) + O(log n)

🫧 Bubble Sort

Compares adjacent elements repeatedly.

O(n²)

⚡ Quick Sort

Divides around a pivot element.

O(n log n)

🔀 Merge Sort

Divides, sorts, then merges halves.

O(n log n)
